# GyampApp

 * Web上のブックマークシステム[Gyamp](http://Gyamp.com/)をMacで使うアプリケーションです
 * Mac画面の上のメニューバー(正式名称はステータスバー)に常駐し、ブックマークにジャンプします
 * 「なうコマンド」で使えるようになっています

## インストール

 * [MacRuby](http://www.macruby.org/)のインストールが必要です
 * XCode3でビルドして下さい

## 使いかた

 * 起動するとMacデスクトップの上部にこういうものが表示されます
